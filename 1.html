HTML

HTML (Hypertext Markup Language) is not a programming language, it is a markup language that allows the browser to know how the web pages we see are structured. This makes programming both complex and simple for developers.

HTML is made up of elements , which wrap and markup different parts of each piece of content in order to be displayed and executed in an appropriate way. tags create hyperlinks that take you to different pages on the web, highlight words, and more.
